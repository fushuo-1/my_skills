# 引用格式样本

本文档展示研究报告中的标准引用格式。

## 行内引用格式

### 单一来源

```markdown
根据国家统计局数据，2024年中国GDP增长率为5.2%[1]，高于市场预期的5.0%[2]。
```

### 多个来源

```markdown
研究表明，AI技术在各行业的渗透率持续提升[1][2][3]，其中金融和医疗行业领先[4][5]。
```

### 连续引用

```markdown
该结论得到多个权威机构的验证[1-3]。同时，市场数据显示...
```

## 参考文献列表格式

### 标准格式

```markdown
## 参考来源

1. [国家统计局 - 2024年国民经济和社会发展统计公报](http://www.stats.gov.cn/) - 官方经济数据 [A]
2. [财经网 - 2024年中国经济数据解读](https://www.caijing.com.cn/) - 市场分析报道 [B]
3. [中国人民银行 - 2024年货币政策执行报告](http://www.pbc.gov.cn/) - 货币政策文件 [A]
4. [国际货币基金组织 - 世界经济展望](https://www.imf.org/) - 全球经济预测 [A]
5. [中汽协 - 2024年汽车产销数据](http://www.caam.org.cn/) - 汽车行业统计 [A]
```

### 格式说明

| 元素 | 说明 | 示例 |
|------|------|------|
| 编号 | 从1开始，按引用顺序排列 | 1. |
| 来源名称 | 简洁明了的名称 | 国家统计局 |
| 链接 | 完整可点击的URL | http://www.stats.gov.cn/ |
| 说明 | 简要描述内容 | 2024年国民经济统计公报 |
| 可靠性等级 | 用[A/B/C/D]标注 | [A] |

## 不同类型来源的引用

### 官方机构（A级）

```markdown
1. [国家统计局](http://www.stats.gov.cn/) - 2024年国民经济统计公报 [A]
2. [中国人民银行](http://www.pbc.gov.cn/) - 货币政策报告 [A]
3. [中国证监会](http://www.csrc.gov.cn/) - 资本市场数据 [A]
```

### 权威媒体（B级）

```markdown
1. [财经网](https://www.caijing.com.cn/) - 2024年经济分析 [B]
2. [财新网](https://www.caixin.com/) - 深度报道 [B]
3. [路透社](https://cn.reuters.com/) - 国际新闻 [B]
```

### 专业平台（C级）

```markdown
1. [36氪](https://36kr.com/) - 科技行业资讯 [C]
2. [东方财富](https://www.eastmoney.com/) - 财经数据 [C]
3. [中国汽车工业协会](http://www.caam.org.cn/) - 行业数据 [A]
```

## 特殊情况处理

### 数据冲突时的引用

```markdown
关于GDP增速，国家统计局公布为5.2%[1]，而国际货币基金组织此前预测为4.8%[2]。实际数据高于预测。
```

### 多个数据一致时的引用

```markdown
多个机构的统计显示，新能源汽车销量均超过700万辆[1][2][3]，数据一致性较高。
```

### 预测数据的引用

```markdown
据国际货币基金组织预测，2025年中国GDP增速将达到4.8%[1]（预测值，存在不确定性）。
```

## APA格式参考（可选）

### 学术格式

```markdown
## 参考文献

国家统计局. (2024). 2024年国民经济和社会发展统计公报. http://www.stats.gov.cn/

Smith, J. (2024). Market Analysis Report. Bloomberg. https://www.bloomberg.com/

International Monetary Fund. (2024). World Economic Outlook. https://www.imf.org/
```

## 引用检查清单

在完成报告前，检查以下项目：

```markdown
引用质量检查清单

完整性
- [ ] 所有引用都有编号
- [ ] 参考文献列表完整
- [ ] 每个引用都有链接

准确性
- [ ] 链接可点击
- [ ] 来源名称准确
- [ ] 可靠性等级正确

一致性
- [ ] 格式统一
- [ ] 编号连续
- [ ] 标注位置恰当
```

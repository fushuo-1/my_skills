# 科技行业研究指南

本文档提供科技行业研究的特定模板和最佳实践。

## 科技行业研究框架

### 研究维度

```markdown
## 核心研究维度

### 1. 技术维度
- 技术发展阶段（萌芽/成长/成熟/衰退）
- 核心技术指标（性能/效率/成本）
- 技术创新趋势（突破点/发展方向）
- 技术竞争格局（专利/标准/生态）

### 2. 市场维度
- 市场规模（容量/增速）
- 市场结构（集中度/细分）
- 市场驱动因素
- 市场风险因素

### 3. 竞争维度
- 主要玩家（厂商/产品）
- 竞争优势（技术/品牌/渠道）
- 竞争策略（定价/差异化）
- 市场份额变化

### 4. 政策维度
- 产业政策支持
- 监管政策变化
- 标准规范要求
- 国际政策影响
```

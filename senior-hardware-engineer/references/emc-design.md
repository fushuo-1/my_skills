# EMC设计规范

## 一、EMC基础理论

### 1. EMC三要素

```
干扰源 → 耦合路径 → 敏感设备

抑制策略：
1. 源头抑制：降低干扰源强度
2. 切断路径：滤波、屏蔽、接地
3. 提高抗扰：增强敏感设备抗扰度
```

### 2. 干扰分类

| 类型 | 频率范围 | 特点 | 主要耦合方式 |
|------|---------|------|-------------|
| 传导发射 | 150kHz-30MHz | 沿电源/信号线 | 导线传导 |
| 辐射发射 | 30MHz-1GHz | 空间传播 | 电磁辐射 |
| 传导抗扰 | 150kHz-80MHz | 电源线注入 | 导线传导 |
| 辐射抗扰 | 80MHz-1GHz | 空间辐射 | 电磁耦合 |

## 二、辐射发射控制

### 1. 主要辐射源

**时钟信号**：
- 晶振及其谐波
- 高速时钟线
- PLL输出

**开关信号**：
- DC-DC开关电源
- PWM输出
- 高速数据线

### 2. 辐射控制措施

#### 时钟电路
```
设计要点：
1. 晶振下方完整地平面
2. 时钟走线最短
3. 串联阻尼电阻（22-33Ω）
4. 并联滤波电容
5. 展频技术（SSC）
```

#### 高速信号
```
设计要点：
1. 差分走线（共模抑制）
2. 阻抗匹配（减少反射）
3. 完整参考平面
4. 避免跨越分割
5. 信号线远离板边
```

#### 开关电源
```
设计要点：
1. 缩小开关环路面积
2. 低ESR/ESL电容
3. 缓冲电路（snubber）
4. 屏蔽电感
5. 输出滤波
```

### 3. 屏蔽设计

**屏蔽效能**：
```
SE = A + R + B (dB)
A: 吸收损耗
R: 反射损耗
B: 多次反射修正

典型值：
- 0.5mm铜板 @ 100MHz: SE ≈ 100dB
- 0.1mm铜箔 @ 100MHz: SE ≈ 80dB
```

**屏蔽罩设计**：
- 接触良好（多点接触）
- 开孔控制（< λ/20）
- 缝隙处理（导电泡棉）
- 接地电阻 < 2.5mΩ

## 三、传导发射控制

### 1. 电源滤波

#### 共模滤波
```
共模扼流圈选择：
- 阻抗 @ 干扰频率 ≥ 需要的衰减
- 饱和电流 > 差模电流
- 漏感影响差模滤波

典型器件：
- 线对线：0.1-10mH
- 线对地：1-100mH
```

#### 差模滤波
```
LC滤波设计：
L = (Vin - Vout) × Vout / (Vin × ΔIL × fsw)
C ≥ ΔIL / (8 × fsw × ΔVout)

典型配置：
- X电容：0.1-1µF
- 差模电感：10-100µH
```

### 2. 滤波器设计

**π型滤波器**：
```
配置：C - L - C
适用：高阻抗源、高阻抗负载
衰减：-40dB/十倍频

设计示例：
- C1, C2: 100nF
- L: 10µH
- 截止频率: fc = 1/π√(LC) ≈ 500kHz
```

**T型滤波器**：
```
配置：L - C - L
适用：低阻抗源、低阻抗负载
衰减：-40dB/十倍频
```

### 3. 接口滤波

**信号线滤波**：
```
滤波配置：
1. RC低通：R=100Ω, C=100pF (低频)
2. 磁珠+电容：FB=600Ω@100MHz, C=1nF
3. 共模扼流圈：针对差分信号

布局原则：
- 滤波器靠近接口
- 滤波前后走线隔离
- 接地最短
```

## 四、ESD防护设计

### 1. ESD模型

**人体模型(HBM)**：
```
参数：C=100pF, R=1.5kΩ
典型测试等级：±2kV, ±4kV, ±8kV, ±15kV

能量计算：
E = 0.5 × C × V²
@ 8kV: E = 0.5 × 100pF × (8kV)² = 3.2mJ
```

### 2. ESD防护器件

| 器件类型 | 响应时间 | 电容 | 钳位电压 | 适用场景 |
|----------|---------|------|---------|----------|
| TVS二极管 | <1ns | 0.5-50pF | 低 | 高速信号 |
| ESD抑制器 | <1ns | <0.5pF | 中 | 超高速信号 |
| 压敏电阻 | ~25ns | 高 | 高 | 电源端口 |
| 放电管 | ~µs | 低 | 高 | 通信端口 |

### 3. ESD设计规则

```
布局原则：
1. ESD器件距接口 < 5mm
2. 信号先经过ESD器件
3. 接地路径最短
4. 多层板使用地过孔

选型原则：
1. VRWM > 工作电压 × 1.2
2. VBR < 最大承受电压
3. VC < 器件耐压 × 0.8
4. 电容不影响信号
```

## 五、EMC测试标准

### 1. 发射测试标准

| 标准 | 适用范围 | 限值等级 |
|------|---------|---------|
| CISPR 32 | 信息技术设备 | Class A/B |
| CISPR 22 | 信息技术设备（旧） | Class A/B |
| FCC Part 15 | 美国 | Class A/B |
| EN 55032 | 欧盟 | Class A/B |

**限值对比**：
```
CISPR 32 Class B（住宅环境）：
传导发射（150kHz-30MHz）：准峰值 66-56dBµV
辐射发射（30MHz-1GHz）：准峰值 40-46dBµV @ 10m

CISPR 32 Class A（商业环境）：
传导发射：放宽10dB
辐射发射：放宽10dB
```

### 2. 抗扰度测试标准

| 测试项目 | 标准 | 典型等级 |
|----------|------|---------|
| ESD | IEC 61000-4-2 | ±4kV接触, ±8kV空气 |
| 快速瞬变 | IEC 61000-4-4 | ±1kV/±2kV |
| 浪涌 | IEC 61000-4-5 | ±1kV/±2kV |
| 辐射抗扰 | IEC 61000-4-3 | 3V/m/10V/m |
| 传导抗扰 | IEC 61000-4-6 | 3V/10V |

### 3. 测试等级选择

**等级推荐**：
```
消费电子：
- ESD: ±4kV接触, ±8kV空气
- EFT: ±1kV
- Surge: ±1kV

工业设备：
- ESD: ±4kV接触, ±8kV空气
- EFT: ±2kV
- Surge: ±2kV

汽车电子：
- ESD: ±8kV接触, ±15kV空气
- EFT: 更严格
- Transient: ISO 7637
```

## 六、EMC设计流程

### 1. 设计阶段EMC考虑

```
原理图阶段：
├── 电源滤波设计
├── 接口滤波设计
├── ESD防护设计
└── 时钟处理

PCB布局阶段：
├── 功能分区
├── 滤波器布局
├── 高速信号处理
└── 接地设计

PCB布线阶段：
├── 电源完整性
├── 信号完整性
├── 参考平面完整性
└── 屏蔽设计

结构设计阶段：
├── 屏蔽罩设计
├── 接地设计
├── 缝隙处理
└── 滤波器安装
```

### 2. EMC问题排查

**辐射发射超标排查流程**：
```
1. 近场探头定位干扰源
   ├── 时钟电路
   ├── 电源电路
   ├── 高速信号
   └── 接口电路

2. 分析耦合路径
   ├── 电源线
   ├── 信号线
   └── 机壳缝隙

3. 针对性整改
   ├── 源头抑制
   ├── 滤波加强
   └── 屏蔽改进
```

# 嵌入式系统设计

## 一、MCU/ARM系统设计

### 1. 电源系统设计

#### 多电源轨设计

**典型ARM SoC电源架构**：
```
输入电源 (5V/12V)
    │
    ├── DC-DC (3.3V) ─── I/O电源
    │
    ├── DC-DC (1.8V) ─── DDR电源
    │
    ├── DC-DC (1.2V) ─── 核心电源
    │
    └── LDO (1.1V)  ─── PLL电源
```

**上电时序要求**：
```
典型时序：
1. 核心电源先上
2. I/O电源后上
3. 时序间隔：1-10ms

时序控制方法：
- RC延时电路
- 电源IC内置时序
- GPIO软件控制
```

#### 功耗管理

**功耗来源分析**：
```
动态功耗：Pd = α × C × V² × f
- α: 活动因子
- C: 负载电容
- V: 工作电压
- f: 工作频率

静态功耗：Ps = V × Ileak
- Ileak: 漏电流
```

**低功耗策略**：
| 策略 | 节能效果 | 实现方式 |
|------|---------|---------|
| 降频 | 20-40% | 动态调频 |
| 降压 | 40-60% | 动态调压 |
| 休眠 | 80-99% | 低功耗模式 |
| 外设关断 | 10-30% | 时钟门控 |

### 2. 时钟系统设计

#### 外部晶振选型

**晶振类型对比**：
| 类型 | 精度 | 成本 | 适用场景 |
|------|------|------|---------|
| 无源晶振 | ±30ppm | 低 | 一般应用 |
| 有源晶振 | ±10ppm | 中 | 高精度应用 |
| TCXO | ±0.5ppm | 高 | 通信设备 |
| OCXO | ±0.01ppm | 很高 | 基站/仪器 |

**负载电容计算**：
```
CL = (C1 × C2) / (C1 + C2) + Cstray

其中：
- C1, C2: 外接电容
- Cstray: PCB寄生电容（2-5pF）

例：晶振CL=12pF
C1 = C2 = 2 × (CL - Cstray)
    = 2 × (12 - 3) = 18pF
```

#### 时钟分配

**时钟树设计**：
```
晶振 → PLL → 分频器 → 各模块
              │
              ├── CPU时钟
              ├── 外设时钟
              └── 总线时钟

设计要点：
1. 时钟源数量最小化
2. PLL配置符合要求
3. 分频比整数优先
4. 时钟切换无毛刺
```

### 3. 复位系统设计

**复位类型**：
| 复位类型 | 触发条件 | 影响 |
|----------|---------|------|
| 上电复位(POR) | 电源上电 | 全系统复位 |
| 看门狗复位 | 程序跑飞 | 全系统复位 |
| 手动复位 | 按键触发 | 全系统复位 |
| 软件复位 | 指令触发 | 部分复位 |
| 低电压复位(BOR) | 电压过低 | 全系统复位 |

**复位电路设计**：
```
典型RC复位电路：
VCC ── R ──┬── RESET
           │
           C
           │
          GND

时间常数：τ = RC
复位时间：T = 3τ ~ 5τ

例：R=10kΩ, C=1µF
T = 5 × 10k × 1µF = 50ms
```

## 二、FPGA系统设计

### 1. FPGA电源设计

**电源轨要求**：
| 电源轨 | 典型电压 | 电流特性 | 纹波要求 |
|--------|---------|---------|---------|
| VCCINT | 0.9-1.2V | 大电流 | < 5% |
| VCCIO | 1.8-3.3V | 中电流 | < 5% |
| VCCAUX | 1.8-2.5V | 小电流 | < 2% |
| VCCBRAM | 0.9-1.2V | 中电流 | < 5% |

**上电时序**：
```
推荐时序：
1. VCCINT / VCCBRAM 先上
2. VCCAUX 随后
3. VCCIO 最后
4. 时序间隔：1-10ms

注意：时序错误可能导致：
- 配置失败
- I/O异常
- 器件损坏
```

### 2. FPGA配置设计

**配置方式对比**：
| 方式 | 存储器 | 速度 | 适用场景 |
|------|--------|------|---------|
| JTAG | - | 慢 | 调试 |
| Master SPI | SPI Flash | 中 | 独立启动 |
| Slave Serial | 外部控制器 | 中 | 级联配置 |
| SelectMAP | 并行 | 快 | 快速加载 |
| Master BPI | 并行Flash | 快 | 大容量 |

**配置引脚处理**：
```
关键配置引脚：
- PROG_B: 配置启动（低有效）
- DONE: 配置完成（高有效）
- INIT_B: 初始化状态（低有效）
- CCLK: 配置时钟
- DIN: 配置数据

设计要点：
1. PROG_B下拉或可控
2. DONE上拉（开漏输出）
3. INIT_B上拉并可作为状态指示
4. 配置时钟走线短
```

### 3. FPGA时钟设计

**时钟输入类型**：
| 类型 | 电平标准 | 频率范围 | 应用 |
|------|---------|---------|------|
| 单端 | LVTTL/LVCMOS | < 100MHz | 低速时钟 |
| 差分 | LVDS | < 500MHz | 高速时钟 |
| 差分 | LVPECL | < 1GHz | 超高速时钟 |

**时钟分配约束**：
```
约束要点：
1. 全局时钟优先（BUFG）
2. 区域时钟次之（BUFR）
3. 局部时钟最后（BUFH）
4. 时钟使能逻辑合理

时序约束示例：
create_clock -period 10 [get_ports clk]
set_input_delay -clock clk 2 [all_inputs]
set_output_delay -clock clk 3 [all_outputs]
```

## 三、DSP系统设计

### 1. DSP电源设计

**电源架构**：
```
输入 (5V/12V)
    │
    ├── DC-DC (1.0V) ─── 核心电源
    │
    ├── DC-DC (1.8V) ─── DDR电源
    │
    ├── DC-DC (3.3V) ─── I/O电源
    │
    └── LDO (1.3V)  ─── PLL电源
```

**电源要求**：
| 电源轨 | 电压精度 | 纹波要求 | 瞬态响应 |
|--------|---------|---------|---------|
| 核心 | ±3% | < 50mV | < 100µs |
| I/O | ±5% | < 100mV | < 1ms |
| PLL | ±2% | < 10mV | < 10µs |

### 2. DSP高速接口

**DDR接口设计**：
```
关键参数：
- 数据速率：高达1600MT/s
- 时钟：差分对，等长匹配
- DQS：数据选通，与DQ等长
- 地址/控制：T型或Fly-by拓扑

布局要点：
1. 靠近DSP放置
2. 完整参考平面
3. 差分等长匹配
4. 去耦电容充足
```

**高速串行接口**：
```
SerDes设计：
- 预加重设置
- 均衡配置
- 眼图优化
- 信号检测

PCB要求：
- 阻抗控制：100Ω差分
- 走线长度匹配
- 参考平面完整
- ESD保护
```

## 四、存储系统设计

### 1. 外部存储器接口

**SDRAM/DDR接口**：
```
信号分组：
- 数据组（DQ[0:n], DQS, DM）
- 地址组（A[0:n], BA, RAS, CAS, WE）
- 控制组（CS, CKE, ODT）
- 时钟组（CLK, CLK#）

设计要点：
1. 数据组组内等长
2. 地址组T型拓扑
3. 时钟差分等长
4. Vref稳定
```

**Flash接口**：
```
NOR Flash：
- 并行接口：地址/数据复用
- SPI接口：标准/双/四线模式

NAND Flash：
- 8位数据总线
- 命令/地址/数据复用
- 需要ECC校验

设计要点：
1. 片选信号干净
2. 读写时序正确
3. 上拉/下拉电阻
```

### 2. 存储器保护

**写保护设计**：
```
硬件保护：
- 写保护引脚控制
- 防止误写入

软件保护：
- 写使能命令序列
- 状态寄存器保护

典型电路：
WP引脚 ── 上拉电阻 ── VCC
           │
        跳线/GPIO ── 可控写保护
```

## 五、接口电路设计

### 1. GPIO设计

**GPIO功能配置**：
```
输入模式：
- 浮空输入
- 上拉输入
- 下拉输入
- 模拟输入

输出模式：
- 推挽输出
- 开漏输出
- 复用功能

设计要点：
1. 上拉/下拉电阻选择
2. 驱动能力匹配
3. 电平转换需求
```

### 2. ADC接口设计

**参考电压设计**：
```
参考源选择：
- 内部参考：精度一般，成本低
- 外部参考：精度高，成本高

关键参数：
- 精度：±0.5% ~ ±0.01%
- 温漂：10-100ppm/°C
- 噪声：< 10µVrms

布局要点：
1. 参考电压走线短
2. 去耦电容靠近引脚
3. 避免数字信号干扰
```

**输入电路设计**：
```
抗混叠滤波：
- RC低通滤波
- 截止频率 ≤ fs/2

驱动电路：
- 运放驱动
- 建立时间充足

保护电路：
- 输入钳位
- 限流电阻
```

### 3. DAC接口设计

**输出电路设计**：
```
输出类型：
- 电压输出
- 电流输出

滤波设计：
- RC低通滤波
- 截止频率根据应用

缓冲设计：
- 运放缓冲
- 驱动能力增强
```

## 六、调试接口设计

### 1. JTAG设计

**JTAG信号**：
```
信号定义：
- TCK: 测试时钟
- TMS: 测试模式选择
- TDI: 测试数据输入
- TDO: 测试数据输出
- TRST: 测试复位（可选）

设计要点：
1. 串联电阻（22-100Ω）
2. 上拉电阻（TMS, TDI）
3. 10pin/20pin连接器
4. 信号走线短
```

### 2. SWD设计

**SWD信号**：
```
信号定义：
- SWCLK: 串行时钟
- SWDIO: 串行数据输入/输出
- SWO: 串行跟踪输出（可选）

优势：
- 引脚少（2-3根）
- 速度适中
- 调试功能完整

设计要点：
1. 串联电阻
2. 上拉电阻
3. 连接器选择
```

### 3. 串口调试

**UART调试接口**：
```
设计要点：
1. TX/RX电平转换（如需）
2. 电阻保护
3. 标准连接器
4. 调试软件支持

常见配置：
- 波特率：115200
- 数据位：8
- 停止位：1
- 校验：无
```

# PCB设计指南

## 一、层叠设计

### 1. 层数选择原则

**信号密度评估**：
- 简单产品（少量信号）：2层
- 消费电子（中等密度）：4层
- 高速产品（大量高速信号）：6层+
- 超高速产品（服务器/通信设备）：8-12层+

**成本考量**：
| 层数 | 相对成本 | 适用场景 |
|------|----------|----------|
| 2层 | 1.0 | 简单控制板、低速产品 |
| 4层 | 1.5-2.0 | 消费电子、普通嵌入式 |
| 6层 | 2.5-3.5 | 高速数字、通信设备 |
| 8层 | 4.0-5.0 | 服务器、高端通信 |

### 2. 叠层结构设计

#### 4层板典型结构
```
Layer 1: Signal (Top)     - 主要元件和信号
Layer 2: GND             - 完整地平面
Layer 3: Power           - 电源平面
Layer 4: Signal (Bottom) - 次要信号
```

**优点**：成本低，有完整参考平面
**缺点**：高速信号布线受限

#### 6层板典型结构
```
Layer 1: Signal (Top)     - 高速信号
Layer 2: GND             - 地参考
Layer 3: Signal (Inner1) - 电源或低速信号
Layer 4: Power/GND       - 电源/地混合
Layer 5: GND             - 地参考
Layer 6: Signal (Bottom) - 高速信号
```

**优点**：高速信号布线空间大，双层参考
**缺点**：成本较高

#### 8层板典型结构
```
Layer 1: Signal (Top)
Layer 2: GND
Layer 3: Signal (Inner1)
Layer 4: GND
Layer 5: Power
Layer 6: Signal (Inner2)
Layer 7: GND
Layer 8: Signal (Bottom)
```

### 3. 材料选择

| 材料 | 介电常数(εr) | 损耗角(tanδ) | 适用频率 |
|------|-------------|-------------|----------|
| FR4 | 4.2-4.8 | 0.02 | < 1GHz |
| 高速FR4 | 4.0-4.5 | 0.015 | < 3GHz |
| Rogers 4350B | 3.48 | 0.0037 | < 10GHz |
| Megtron 6 | 3.6 | 0.006 | < 5GHz |

**选择原则**：
- 低速产品：标准FR4
- 高速产品（>1GHz）：高速板材或混压
- 射频产品：Rogers等低损耗材料

## 二、布局原则

### 1. 功能分区

**典型布局分区**：
```
┌─────────────────────────────────────┐
│  电源区  │  数字区  │  模拟区  │  接口区  │
│ (输入端) │ (核心MCU)│ (传感器) │ (连接器) │
└─────────────────────────────────────┘
```

**分区原则**：
- 电源输入端靠近板边
- 高速器件靠近板中心
- 敏感模拟电路远离噪声源
- 接口器件靠近板边

### 2. 器件放置

**核心器件优先**：
1. 放置主芯片（MCU/SoC）
2. 放置晶振等时钟器件
3. 放置存储器等高速器件
4. 放置电源模块
5. 放置接口器件

**关键器件间距**：
- 晶振距离MCU：< 20mm
- 去耦电容距离IC：< 5mm
- 高速器件间：考虑串扰

### 3. 热设计布局

**热管理策略**：
- 大功率器件分散布局
- 预留散热器空间
- 热敏感器件远离热源
- 考虑风道方向

## 三、布线规则

### 1. 通用布线规则

**线宽线距**：
| 电流(A) | 线宽(mm) @ 1oz铜厚 |
|---------|-------------------|
| 0.5 | 0.15 |
| 1.0 | 0.3 |
| 2.0 | 0.6 |
| 3.0 | 1.0 |

**基本规则**：
- 最小线宽：0.1mm (4mil) - 可制造性考虑
- 最小间距：0.1mm (4mil) - 可制造性考虑
- 3W原则：线间距 ≥ 3倍线宽，降低串扰
- 20H原则：电源平面边缘距地平面边缘 ≥ 20倍层间距

### 2. 电源布线

**设计要点**：
- 电源路径短而宽
- 电源层与地层相邻
- 载流能力计算：1oz铜厚约1A/mm
- 考虑温升：实际载流能力需降额

**过孔设计**：
- 电源过孔并联增加载流
- 单过孔载流：约1A @ 0.3mm孔径
- 电源过孔数量 = 最大电流 / 单过孔载流

### 3. 高速信号布线

#### 差分信号
**设计规则**：
- 差分对等长匹配：< 5mil
- 差分对间距一致
- 差分阻抗控制：90Ω±10% 或 100Ω±10%

**差分阻抗公式**：
```
Zdiff ≈ 2 × Z0 × (1 - 0.48 × e^(-0.96×S/h))
Z0: 单端阻抗
S: 差分对间距
h: 到参考平面距离
```

#### 时钟信号
**设计规则**：
- 时钟走线下方参考平面完整
- 避免跨越分割平面
- 时钟走线避免与高速信号平行走线
- 蛇形走线等长匹配

#### DDR信号
**设计规则**：
- 数据线(DQ)组内等长：< 10ps
- 地址/控制线等长：< 20ps
- 时钟差分对等长：< 5mil
- 走线拓扑：T型或Fly-by

### 4. 模拟信号布线

**设计要点**：
- 模拟电路独立区域
- 模拟地与数字地分离（单点连接）
- 输入输出信号避免交叉
- 屏蔽敏感信号

## 四、接地设计

### 1. 接地策略

**接地方式选择**：
| 频率范围 | 推荐接地方式 |
|----------|-------------|
| < 1MHz | 单点接地 |
| 1-10MHz | 多点接地 |
| > 10MHz | 多点接地 |

### 2. 地平面设计

**设计原则**：
- 地平面尽量完整
- 避免地平面分割
- 过孔数量控制（影响回流）
- 回流路径最短原则

**分割处理**：
- 必须分割时用磁珠/0Ω电阻桥接
- 分割处避免信号线跨越
- 跨分割信号需要桥接电容

### 3. 混合信号接地

**AGND与DGND**：
```
方案1：物理分离，单点连接
方案2：统一地平面，布局分区
方案3：芯片下方单点连接（推荐）
```

## 五、过孔设计

### 1. 过孔类型

| 过孔类型 | 尺寸范围 | 成本 | 应用场景 |
|----------|---------|------|----------|
| 通孔 | 孔径0.3-0.6mm | 低 | 一般信号 |
| 盲孔 | 孔径0.1-0.2mm | 高 | 高密度板 |
| 埋孔 | 孔径0.1-0.2mm | 高 | 多层板 |

### 2. 过孔寄生参数

**寄生电感**：
```
L ≈ 5.08h × [ln(4h/d) + 1] (nH)
h: 过孔长度(英寸)
d: 过孔直径(英寸)
```

**寄生电容**：
```
C ≈ 1.41εrTD1/(D2-D1) (pF)
εr: 介电常数
T: 板厚
D1: 焊盘直径
D2: 阻焊直径
```

### 3. 过孔设计规则

- 电源过孔：多过孔并联
- 高速信号：考虑过孔残桩影响
- 差分信号：过孔对称放置
- BGA扇出：遵循BGA间距

## 六、设计验证

### 1. DFM检查

**检查项目**：
- 最小线宽线距
- 最小焊盘间距
- 铜皮间距
- 阻焊桥宽度
- NPTH孔到线路距离

### 2. 信号完整性检查

**检查项目**：
- 阻抗控制
- 差分等长
- 参考平面完整性
- 回流路径

### 3. 电源完整性检查

**检查项目**：
- 电源载流能力
- PDN阻抗
- 去耦电容布局
- IR压降

### 4. EMC预评估

**检查项目**：
- 地平面完整性
- 高速信号屏蔽
- 接口滤波
- 时钟走线处理

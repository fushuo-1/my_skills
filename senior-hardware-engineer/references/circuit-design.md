# 电路设计详解

## 一、模拟电路设计

### 1. 电源电路设计

#### LDO设计要点

**选型参数**：
- 输入电压范围
- 输出电压精度
- 输出电流能力
- 压差(Vdropout)
- PSRR（电源抑制比）
- 输出噪声
- 静态电流(Iq)

**设计公式**：
```
输出纹波 = 负载电流变化 × 输出电容ESR
功率损耗 = (Vin - Vout) × Iout
热阻计算：Tj = Ta + Pd × θja
```

**实例**：TPS7A47低噪声LDO
- 输出噪声：3.8µVrms (10Hz-100kHz)
- PSRR：72dB @ 1kHz
- 压差：200mV @ 1A

#### DC-DC转换器设计

**Buck转换器设计**：
```
电感选择：
L = (Vin - Vout) × Vout / (Vin × ΔIL × fsw)

输出电容选择：
Cout ≥ ΔIL / (8 × fsw × ΔVout)

输入电容选择：
Cin ≥ Iout × D × (1-D) / (fsw × ΔVin)
```

**设计要点**：
- 电感饱和电流 > 峰值电流 × 1.2
- 输出电容ESR影响纹波
- 输入电容需靠近IC放置
- 反馈网络布局影响稳定性

### 2. 滤波电路设计

#### RC低通滤波器
```
截止频率：fc = 1 / (2πRC)
衰减斜率：-20dB/十倍频
```

#### 有源滤波器设计

**Sallen-Key二阶低通**：
```
截止频率：fc = 1 / (2π√(R1R2C1C2))
Q值：Q = √(R1R2C1C2) / (C1(R1+R2))
```

**设计要点**：
- 选择高Q值电容（NPO/C0G）
- 运放带宽 > 10 × fc
- 考虑元件容差对Q值的影响

### 3. 放大电路设计

#### 运放选型考虑因素
- 增益带宽积(GBW)
- 压摆率(SR)
- 输入失调电压(Vos)
- 输入偏置电流(Ib)
- 噪声密度
- 电源电压范围

#### 反馈稳定性分析
```
相位裕度检查：
1. 开环增益曲线与噪声增益曲线交点
2. 交点处相位 > 180° + 45°（相位裕度）
3. 典型要求：相位裕度 ≥ 45°
```

## 二、数字电路设计

### 1. 电平标准

| 标准 | VCC | VOH(min) | VOL(max) | VIH(min) | VIL(max) |
|------|-----|----------|----------|----------|----------|
| TTL | 5V | 2.4V | 0.4V | 2.0V | 0.8V |
| CMOS | 5V | 4.9V | 0.1V | 3.5V | 1.5V |
| LVCMOS | 3.3V | 2.4V | 0.4V | 2.0V | 0.8V |
| LVTTL | 3.3V | 2.0V | 0.4V | 2.0V | 0.8V |

### 2. 时序分析

**建立时间与保持时间**：
```
建立时间余量：
Tsetup_margin = Tclk - Tco_max - Tdata_max - Tsetup

保持时间余量：
Thold_margin = Tco_min + Tdata_min - Thold
```

**设计要求**：
- 建立时间余量 > 0
- 保持时间余量 > 0
- 推荐：余量 ≥ 10% 时钟周期

### 3. 接口电路设计

#### I2C接口
- 上拉电阻计算：Rp = (VCC - VOL) / IOL
- 典型值：4.7kΩ @ 100kHz, 2.2kΩ @ 400kHz
- 总线电容 < 400pF

#### SPI接口
- 时钟极性(CPOL)和相位(CPHA)配置
- 最大时钟频率受PCB走线长度影响
- 长距离传输考虑增加驱动器

#### UART接口
- 波特率误差 < 2%
- 电平转换考虑（如MAX232）
- 长距离传输考虑RS485

## 三、高速电路设计

### 1. 传输线基础

**特性阻抗**：
```
微带线：Z0 = 87/√(εr+1.41) × ln(5.98h/(0.8w+t))
带状线：Z0 = 60/√εr × ln(4h/(0.67π(0.8w+t)))
```

**反射控制**：
```
反射系数：Γ = (ZL - Z0)/(ZL + Z0)
目标：|Γ| < 0.1
```

### 2. 端接方式

| 端接方式 | 优点 | 缺点 | 适用场景 |
|----------|------|------|----------|
| 串联端接 | 功耗低 | 只适合单负载 | 点对点传输 |
| 并联端接 | 匹配精确 | 功耗大 | 高速信号 |
| 戴维南端接 | 匹配好 | 功耗大 | 差分信号 |
| AC端接 | 无直流功耗 | 仅适合周期信号 | 时钟信号 |

### 3. 差分信号设计

**差分阻抗**：
```
Zdiff ≈ 2 × Z0 × (1 - 0.48 × exp(-0.96 × S/h))
其中：S为差分对间距，h为介质厚度
```

**设计规则**：
- 差分对等长误差 < 5mil
- 差分对间距一致
- 避免差分对跨越参考平面分割
- 优先生成参考平面完整区域

### 4. DDR设计要点

**DDR4关键参数**：
- 数据速率：1600-3200 MT/s
- VDD：1.2V
- ODT配置：34Ω/40Ω/48Ω/60Ω/120Ω
- Vref：VDD/2

**布局要求**：
- 数据线等长匹配：< 10ps
- 地址/控制线T型拓扑
- 时钟差分对等长：< 5mil
- 去耦电容靠近电源引脚

## 四、电源完整性设计

### 1. PDN阻抗设计

**目标阻抗计算**：
```
Ztarget = ΔV / ΔI

例：1.2V电源，允许纹波5%，瞬态电流3A
Ztarget = 1.2 × 0.05 / 3 = 20mΩ
```

### 2. 去耦电容选择

**容值选择策略**：
- 大电容（10µF-100µF）：低频去耦，VRM响应
- 中电容（0.1µF-1µF）：中频去耦
- 小电容（1nF-100nF）：高频去耦

**ESL影响**：
```
自谐振频率：fSRF = 1 / (2π√(L × C))
工作频率应 < fSRF
```

### 3. 去耦电容布局

**布局原则**：
- 距离IC电源引脚 < 5mm
- 过孔靠近电容焊盘
- 电源和地过孔成对放置
- 多层电容垂直堆叠

## 五、保护电路设计

### 1. ESD防护

**防护器件选择**：
- TVS二极管：响应快，钳位电压低
- ESD抑制器：电容低，适合高速信号
- 压敏电阻：成本低，响应慢

**设计要点**：
- ESD器件靠近接口放置
- 走线最短原则
- 接地路径最短

### 2. 过流保护

**保护方案**：
- 保险丝：一次性保护
- PTC：可恢复保护
- 电子保险丝：精确控制

**选型考虑**：
- 额定电流 > 工作电流 × 1.5
- 动作时间与被保护器件匹配
- 考虑浪涌电流

### 3. 过压保护

**TVS选型**：
```
VRWM > 正常工作电压
VBR < 最大承受电压
VC < 器件最大耐压
```
